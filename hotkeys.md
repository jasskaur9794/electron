# Finder — Hotkeys

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+X` | Capture screenshot & solve with AI |
| `Ctrl+Shift+D` | Toggle overlay visibility (show/hide) |
| `Ctrl+Shift+↑` | Show previous answer |
| `Ctrl+Shift+↓` | Show next answer |
| `Ctrl+Shift+K` | Scroll answer content up |
| `Ctrl+Shift+J` | Scroll answer content down |
| `Ctrl+Shift+C` | Clear all answer history |
| `Ctrl+Shift+Q` | Quit Finder |

## CLI Commands

| Command | Purpose |
|---|---|
| `npm start` | Launch in background (detached, silent) |
| `npm run start:attached` | Launch in terminal (for development) |
| `npm run start:verbose` | Launch with debug logging |
| `npm run kill` | Kill all running Finder instances |
